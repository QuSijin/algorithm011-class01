3.二叉堆实现细节：
	1)二叉堆一般都通过“数组”来实现
	2)假设“第一个元素”在数组中的索引为0的话，则父结点和子结点的位置关系如下
	索引为i的左孩子的索引是(2*i+1)
	索引为i的左孩子的索引是(2*i+2)
	索引为i的父结点的索引是floor((i-1)/2)

	2)树中任意节点的值总是>=真子结点的值
